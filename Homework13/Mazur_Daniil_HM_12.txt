Сделать Fork репозитория

ID: MF-11
Требование: создать ветвление проекта
Модуль: Fork
Имя: Создание ветвления проекта в собственном репозитории github
Предусловие/Setup: Созданная учетная запись на github.com
Шаги:
1. 	Запустить браузер. 
2.	 Ввести в строке поиска браузера  github.com.
3.	Открыть github.com.
4. Нажать «Sing in».
5. Ввести в окне «Username or email addresse» данные зарегистрированного пользователя.
6. Ввести в окне «Password» пароль зарегистрированного пользователя.
7.	Нажать «Sing in».
8.	Войти в свою учетную запись. 
9. Ввести имя требуемого для поиска пользователя вверху в поисковой строке.
10. Нажать «All github».
11.	Нажать «User».
12.	Выбрать из списка требуемого пользователя.
13.	Перейти по ссылке на страницу выбранного пользователя.
14.	Нажать вкладку «Repositiries».
15.	Выбрать необходимый репозиторий.
16.	Перейти по ссылке выбранного репозитория.
17.	Нажать «Fork» в правом верхнем углу.
18.	Нажать «Create fork».

Ожидаемый результат: Создание копии выбранного репозитория в своём аккаунте.
Приоритет: Medium
Постусловие/Teardown: внесение любых изменений в код в вашем разветвленном репозитории.








Закрытие Pull Request ветки «test» в ветку «master»

ID: MB-12
Требование: закрыть ветку Pull Request
Модуль: Repositories
Имя:  закрытие Pull Request ветки «test» в ветку «master»
Предусловие/Setup: отправить ветку «test» в в удаленный репозиторий  github зарегистрированного пользователя

Шаги:
1. Нажать вкладку «Repositories» на странице профиля зарегистрированного пользователя
2. Выбрать необходимый репозиторий.
3. Кликнуть вкладку «Pull request».
4. Кликнуть по кнопке «New pull request».
5. Выбрать ветку «test».
6. Кликнуть на кнопку «Compare & pull request».
7. Выбрать кнопку «Create pull request».
8. Кликнуть по вкладке «Conversation».
9. Выбрать в выпадающем списке «Rebase and merge».
10.	Кликнуть «Confirm merge»

Ожидаемый результат: ветка «test»  объединилась с веткой «master»
Приоритет: high
Постусловие/Teardown: удаление ветки «test»
















Изменение Bio в профиле пользователя
ID: СB-13
Требование:  изменить  Bio в профиле пользователя
Модуль: Профиль
Имя: изменение сведений  профиле пользователя,  
Предусловие/Setup: авторизовать учетную запись на github.com

Шаги:
1. Нажать иконку профиля в верхнем правом углу на странице пользователя.
2. Выбрать запись «Your profile» в выпадающем списке.
3.	Нажать  «Edit profile», располагающейся  слева сбоку страницы профиля.
4.	Заполнить в предлагаемом окне Bio необходимые изменения.
 5. 	Нажать «Save».
 
Ожидаемый результат: внесенные Bio изменения сохранились.
Приоритет: Medium.
Постусловие/Teardown: обновление страницы профиля.
